# מחשבון חלבון וקלוריות 💪

**אפליקציית PWA לעקיבה אחרי חלבון וקלוריות יומיים — עובדת גם ללא אינטרנט**

🔗 **[פתח את האפליקציה](https://ben9730.github.io/macro-calculator/)**

---

## התקנה על הטלפון (iPhone)

1. פתח את הקישור בـ **Safari**: `https://ben9730.github.io/macro-calculator/`
2. לחץ על כפתור השיתוף ↑
3. בחר **"הוסף למסך הבית"**
4. האפליקציה תופיע כאייקון — עובדת כמו אפליקציה רגילה, גם בלי אינטרנט

---

## שימוש

### 📊 היום
- רואים את סך החלבון והקלוריות שנאכלו היום
- **יעד חלבון** — ניתן לשנות ישירות (ברירת מחדל: 150 גרם)
- **הוספה מהירה** — לחיצה אחת על מאכל שמור מוסיפה אותו ליום
- לחיצה על × מסירה ארוחה
- "נקה את היום" — מאפס את כל הארוחות של היום

### ＋ הוספה
שני אופני הזנה:

**📷 סריקת תווית אוטומטית** (צריך מפתח API)
- לחץ ⚙ בפינה → הכנס מפתח Anthropic API → שמור
- לחץ "צלם או העלה תווית" → צלם את טבלת "ל-100 גרם"
- הערכים מתמלאים אוטומטית

**✍️ הזנה ידנית**
- מלא את הערכים התזונתיים ל-100 גרם בטבלה האדומה
- בחר כמות (100 / 150 / 200 גרם) או הקלד בעצמך
- לחץ **"＋ הוסף ליום"** — מוסיף לרשימת היום
- לחץ **"★ שמור כמאכל"** — שומר לשימוש חוזר

### ★ מאכלים
- רשימת כל המאכלים השמורים
- **＋ ליום** — מוסיף מיד בכמות ברירת המחדל ועובר להיום
- **ערוך כמות** — טוען את המאכל בטאב ההוספה לעריכה
- 🗑 — מוחק את המאכל

---

## סריקת תוויות — הגדרת מפתח API

הסריקה האוטומטית משתמשת ב-Claude AI (Anthropic).  
המפתח נשמר **רק במכשיר שלך** ולא נשלח לשום שרת מלבד Anthropic.

1. היכנס לـ [console.anthropic.com](https://console.anthropic.com)
2. צור מפתח API חינמי
3. פתח ⚙ באפליקציה → הכנס את המפתח → שמור
